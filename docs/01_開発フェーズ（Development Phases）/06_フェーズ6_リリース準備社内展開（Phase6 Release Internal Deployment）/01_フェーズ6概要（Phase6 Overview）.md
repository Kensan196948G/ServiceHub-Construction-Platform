# フェーズ6: リリース準備・社内展開 概要

## フェーズ目標

フェーズ6は、2026年10月2日の社内リリースに向けた最終準備フェーズである。UAT（ユーザー受入テスト）の実施、社内デプロイ、ユーザー研修、運用移行を完了し、ServiceHub Construction Platform を社内に安全・スムーズに展開する。

| 項目 | 内容 |
|------|------|
| フェーズ番号 | Phase 6 |
| 期間 | 2026/09/01〜2026/10/02（約32日間） |
| 作業時間 | 8時間/日（合計約256時間） |
| 主要テーマ | UAT・社内デプロイ・研修・運用移行 |
| 前提 | フェーズ5（品質保証）の完了 |

---

## フェーズ目標

1. **UAT完了**：実際のエンドユーザーによる受入テストの合格
2. **本番デプロイ**：安全な本番環境へのデプロイと動作確認
3. **ユーザー研修**：全ユーザーへのシステム操作研修の実施
4. **運用移行**：既存業務からシステムへの円滑な移行
5. **サポート体制確立**：運用開始後のサポート体制の整備
