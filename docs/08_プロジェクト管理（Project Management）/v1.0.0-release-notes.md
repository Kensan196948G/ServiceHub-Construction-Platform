# ServiceHub Construction Platform v1.0.0-rc.1 リリースノート

**リリース日**: 2026-04-03  
**バージョン**: v1.0.0-rc.1（リリース候補 1）  
**ステータス**: UAT 実施前・社内レビュー段階

---

## 概要

建設業向け統合業務プラットフォーム **ServiceHub Construction Platform** の初回リリース候補版です。  
工事案件管理から ITSM・ナレッジ管理・AI 検索まで、建設現場の業務フローを一元化する 7 モジュールを実装しています。

---

## 実装済み機能一覧（7 モジュール）

### 1. 認証・認可モジュール
- JWT 認証（HS256 / アクセストークン 15 分 / リフレッシュトークン 7 日）
- RBAC 認可（ADMIN / PROJECT_MANAGER / SITE_SUPERVISOR / COST_MANAGER / IT_OPERATOR / VIEWER）
- SoD（職務分離）実装（変更承認: ADMIN のみ）
- フロントエンド: LoginPage / Zustand authStore（JWT 永続化）/ axios interceptor

### 2. 工事案件管理モジュール
- 案件 CRUD（作成・取得・更新・削除）
- ステータス管理（計画中 / 進行中 / 完了 / 中断）
- 予算管理
- フロントエンド: ProjectsPage（一覧・ページネーション・ステータスバッジ）/ ProjectDetailPage

### 3. 日報管理モジュール
- 日報作成・提出・承認ワークフロー
- プロジェクト別一覧・ステータス管理
- フロントエンド: DailyReportsPage（新規作成モーダル・ステータス管理）

### 4. 安全・品質管理モジュール
- 安全確認チェックリスト（日次点検記録）
- 品質検査記録（検査結果管理）
- フロントエンド: SafetyPage（安全チェック / 品質検査タブ切替）

### 5. 原価・工数管理モジュール
- コスト記録 CRUD
- 予実対比サマリー API（予算 / 実績 / 差異）
- フロントエンド: CostPage（サマリーカード・原価記録テーブル）

### 6. ITSM 運用管理モジュール（ISO20000 準拠）
- インシデント管理（INC 番号採番 / 優先度 / 解決管理）
- 変更要求管理（CHG 番号採番 / 変更承認ワークフロー / SoD）
- フロントエンド: ItsmPage（インシデント / 変更要求タブ・優先度カラーバッジ）

### 7. ナレッジ管理・AI 支援モジュール
- ナレッジ記事 CRUD（カテゴリ / タグ / 公開管理）
- AI 検索 API（OpenAI gpt-4o-mini / フォールバックキーワード検索）
- AI 検索ログ（監査・改善用）
- フロントエンド: KnowledgePage（記事一覧・AI 検索・カテゴリフィルター）

## 既知の制限事項

1. **UAT 未実施**: エンドツーエンドのユーザー受入テストはまだ実施されていません
2. **MinIO 写真管理未テスト**: docker-compose 統合環境での写真アップロード機能の動作確認が未完了
3. **E2E テスト未整備**: Playwright 等による自動 E2E テストスイートが未実装
4. **本番 SSL 証明書未設定**: `docker-compose.prod.yml` を使用する際は別途 SSL 証明書の設定が必要
5. **OpenAI API キー必須**: AI 検索機能は `OPENAI_API_KEY` 環境変数が設定されていない場合、キーワード検索にフォールバック
6. **パフォーマンステスト**: Locust による負荷試験は計画のみで未実施
