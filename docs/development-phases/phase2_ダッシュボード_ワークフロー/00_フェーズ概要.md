3. **通知機能（最小限）** — 承認依頼・承認完了の画面内通知

## 技術選定

| 要素 | 技術 | 理由 |
|------|------|------|
| グラフ | Recharts（既存依存） | 既にインストール済み、React対応 |
| 承認ステータス | PostgreSQL ENUM + 遷移テーブル | 監査証跡の確保 |
| 通知 | React Query + Zustand | WebSocket不要で実装可能 |
