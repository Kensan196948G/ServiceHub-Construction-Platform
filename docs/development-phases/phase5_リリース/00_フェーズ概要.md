## フェーズ目標

1. **本番環境構築** — 社内サーバーに Docker Compose で全サービスをデプロイ
2. **データ移行** — 初期データ投入、マスタデータ設定
3. **UAT** — 主要ユーザーによる受入テスト
4. **運用手順整備** — 起動/停止/バックアップ/障害対応手順
5. **ユーザーマニュアル** — 各機能の操作ガイド
6. **社内公開** — 告知・アカウント配布・サポート体制
