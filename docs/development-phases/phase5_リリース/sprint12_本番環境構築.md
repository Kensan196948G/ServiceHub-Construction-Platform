# Sprint 12: 本番環境構築＋データ移行

## 概要

| 項目 | 内容 |
|------|------|
| 期間 | 2026-09-07 〜 2026-09-20 |
| 工数 | 32時間（4日 × 8h） |
| テーマ | 社内オンプレサーバーへのデプロイと本番環境の安定化 |

## 前提条件

- 社内サーバーが用意されていること（OS: Linux, Docker対応）
- ネットワーク設定（社内LAN内のIP/DNS割当済み）
- SSH アクセス可能

## STABLE判定条件

- 全サービスが本番サーバーで稼働
- HTTPS でアクセス可能
- E2E ヘルスチェック全項目 PASS
- 監視ダッシュボードにメトリクス表示
- バックアップ/リストアが動作確認済み
- **STABLE N=5（本番環境のため）**
