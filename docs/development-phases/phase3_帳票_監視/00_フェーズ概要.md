2. **監視基盤** — Prometheus + Grafana + Node Exporter + Alertmanager をDocker Composeに統合
3. **アプリケーションメトリクス** — FastAPIにPrometheusメトリクス公開エンドポイント追加

## 技術選定

| 要素 | 技術 | 理由 |
|------|------|------|
| PDF生成 | WeasyPrint or reportlab | Python純正、日本語フォント対応 |
| Excel生成 | openpyxl | 軽量、書式制御可能 |
| 監視 | Prometheus + Grafana | docs/05_運用設計 で設計済み |
| メトリクス | prometheus-fastapi-instrumentator | FastAPI自動計装 |
| アラート | Alertmanager | Prometheus標準 |
