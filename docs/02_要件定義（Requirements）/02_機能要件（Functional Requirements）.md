# 機能要件

## 機能要件一覧

### 1. 認証・認可（AUTH）

| 要件ID | 要件名 | 詳細 | 優先度 |
|-------|-------|------|-------|
| AUTH-001 | ユーザー認証 | メール・パスワードによるログイン | 必須 |
| AUTH-002 | JWT認証 | JWTトークンによるAPI認証 | 必須 |
| AUTH-003 | OAuth2.0 | OAuth2.0 Authorization Code Flow | 必須 |
| AUTH-004 | MFA | TOTP（Google Authenticator互換）によるMFA | 必須 |
| AUTH-005 | RBAC | ロールベースアクセス制御の実装 | 必須 |
| AUTH-006 | セッション管理 | リフレッシュトークンによるセッション管理 | 必須 |
| AUTH-007 | パスワードリセット | メールによるパスワードリセット | 必須 |
| AUTH-008 | アカウントロック | ログイン失敗5回でロック（30分） | 必須 |

### 2. 工事案件管理（PROJECT）

| 要件ID | 要件名 | 詳細 | 優先度 |
|-------|-------|------|-------|
| PRJ-001 | 案件作成 | 工事案件の新規登録（コード・名称・工期等） | 必須 |
| PRJ-002 | 案件検索 | 名称・コード・ステータス・期間での検索 | 必須 |
| PRJ-003 | 案件詳細表示 | ダッシュボード形式の案件詳細ビュー | 必須 |
| PRJ-004 | 案件更新 | 案件情報の編集・更新 | 必須 |
| PRJ-005 | ステータス管理 | 計画中→進行中→完了等のステータス管理 | 必須 |
| PRJ-006 | メンバー管理 | 案件メンバーの追加・削除・ロール設定 | 必須 |
| PRJ-007 | 進捗管理 | 工程進捗率の入力・グラフ表示 | 必須 |
| PRJ-008 | 地図表示 | 工事場所の地図表示（Google Maps連携） | 推奨 |

### 3. 日報管理（REPORT）

| 要件ID | 要件名 | 詳細 | 優先度 |
|-------|-------|------|-------|
| RPT-001 | 日報作成 | テンプレートベースの日報入力 | 必須 |
| RPT-002 | 工数記録 | 作業種別・人員・時間の記録 | 必須 |
| RPT-003 | 写真添付 | 日報への作業写真添付（最大10枚） | 必須 |
| RPT-004 | 日報提出 | 承認者への日報提出 | 必須 |
| RPT-005 | 承認フロー | 多段階承認（監督→所長）フロー | 必須 |
| RPT-006 | 差し戻し | 修正依頼・再提出 | 必須 |
| RPT-007 | PDF出力 | 日報のPDFエクスポート | 必須 |
| RPT-008 | AI補完 | AIによる作業概要の自動生成 | 推奨 |
| RPT-009 | 月次集計 | 月次の作業実績集計レポート | 推奨 |

### 4. 写真・資料管理（MEDIA）

| 要件ID | 要件名 | 詳細 | 優先度 |
|-------|-------|------|-------|
| MED-001 | ファイルアップロード | 写真・資料のアップロード（最大50MB） | 必須 |
| MED-002 | サムネイル生成 | 画像サムネイルの自動生成 | 必須 |
| MED-003 | タグ管理 | フリータグによる写真分類 | 必須 |
| MED-004 | ファイル検索 | 名称・タグ・日付・案件での検索 | 必須 |
| MED-005 | ダウンロード | 単体・一括ダウンロード（ZIP） | 必須 |
| MED-006 | アクセス制御 | RBACによるファイルアクセス制御 | 必須 |

### 5. 安全・品質管理（SAFETY）

| 要件ID | 要件名 | 詳細 | 優先度 |
|-------|-------|------|-------|
| SFT-001 | 安全点検 | 安全点検チェックリストの記録 | 必須 |
| SFT-002 | ヒヤリハット | 潜在的危険事象の報告・管理 | 必須 |
| SFT-003 | 品質検査 | 品質検査の実施・記録 | 必須 |
| SFT-004 | 是正処置 | 不適合事象の是正・予防処置管理 | 必須 |
| SFT-005 | 安全KPI | 無災害日数・点検実施率の管理 | 推奨 |

### 6. 原価・工数管理（COST）

| 要件ID | 要件名 | 詳細 | 優先度 |
|-------|-------|------|-------|
| CST-001 | 予算登録 | 費目別予算の登録・管理 | 必須 |
| CST-002 | 実績原価入力 | 日次実績原価の入力・管理 | 必須 |
| CST-003 | 工数集計 | 日報からの工数自動集計 | 必須 |

| CST-006 | 予算アラート | 予算超過時のアラート通知 | 推奨 |

### 7. ITSM管理（ITSM）

| 要件ID | 要件名 | 詳細 | 優先度 |
