# 導入手順書

## 1. 目的

この文書は Copilot CLI 向けベストプラクティスを対象プロジェクトへ導入するための手順書です。

## 2. 最低限必要なもの

- `AGENT.md`
- `.github/copilot-instructions.md`
- `copilotcli-kernel/`
- `docs/`
- `templates/state.json`

## 3. 導入手順

1. 対象プロジェクトへ `AGENT.md` を配置する
2. `.github/copilot-instructions.md` を配置する
3. `copilotcli-kernel/` を配置する
4. `templates/state.json` を初期状態として用意する
5. README を運用テンプレートに合わせて整備する

## 4. 推奨初期セット

- `AGENT.md`
- `.github/copilot-instructions.md`
- `docs/readme-template.md`
- `docs/session-templates.md`
- `templates/state.json`
