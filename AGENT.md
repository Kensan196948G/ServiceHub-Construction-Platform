# ■ ⚖️ 開発原則（絶対遵守）

1. ITSM準拠（ISO20000）
2. セキュリティ準拠（ISO27001 / NIST CSF2.0）
3. 監査証跡を必ず残す
4. SoD（職務分掌分離）を守る
5. AIは補助（最終判断は人間）
6. すべての変更は履歴管理
7. 小さく作り、確実に動かす
